# Personal Portfolio Website

This is my personal portfolio website built to showcase my skills, education, experience, projects, and contact information. The portfolio was created using React, Vite, HTML, CSS, JavaScript, Git, and GitHub Pages.

## Live Website

[View My Portfolio](https://SabitAbdul.github.io/my-portfolio/)

## About This Project

This portfolio website is designed as a professional online profile where visitors can learn more about me, my technical skills, my educational background, my development experience, and the projects I have built.

The goal of this project is to practice modern frontend development while creating a useful website that I can share on GitHub, LinkedIn, and job applications.

## Portfolio Sections

### Home

The home section introduces who I am and what I do. It gives visitors a quick first impression of my portfolio and includes buttons to view my projects or contact me.

### About

The about section gives a short summary of my background, interests, and goals as a developer. It explains my motivation for learning web development and building real-world projects.

### Skills

The skills section lists the technologies and tools I am learning and using, including HTML, CSS, JavaScript, React, Git, GitHub, VS Code, and responsive design.

### Education

The education section highlights my academic background, degree program, school, and learning journey. It shows the foundation that supports my technical and professional growth.

### Experience

The experience section describes my hands-on practice with frontend development. It includes my work with projects, coding practice, GitHub, and building responsive websites.
